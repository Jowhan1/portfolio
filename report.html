About my exposure to web

Learning HTML and CSS has been a highly enjoyable and satisfying journey for me. Under the guidance of my instructor, Chris, I was able to grasp the fundamentals with ease. Initially, I felt apprehensive about creating a complete website independently, but working with these languages turned out to be surprisingly fun. As I progressed through lessons and practice exercises, my confidence increased, and I discovered a genuine passion for web development—a realization that truly amazed me.

When designing my website, I opted for a minimalist style, incorporating different shades of blue, which happens to be my favorite color. This color palette perfectly matched the mood I envisioned for the site. To personalize it further, I explored Google Fonts to select a font that aligned with my aesthetic, and I even created a custom logo for the homepage using Wix. Looking back, I’m genuinely proud of how my first web design project turned out!.
